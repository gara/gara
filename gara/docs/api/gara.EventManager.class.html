EventManager

gara.EventManager

public class EventManager

EventManager is used to store all event listeners throughout the document. This helps to keep all listeners stored in one point and also pretend memory leaks by releasing all listeners at the unload event.

Author:
Thomas Gossmann
See Also:
http://ajaxcookbook.org/event-handling-memory-leaks/

Method Summary

void _unregisterAllEvents() Removes all stored listeners on the page unload.
Event addListener(HTMLElement domNode, DOMString type, Object|Function listener) Adds a listener to a specified domNode and store the added event in the event manager.
void handleEvent() handleEvent is used to catch the unload-event of the window and pass it to _unregisterAllEvents() to free up memory.
